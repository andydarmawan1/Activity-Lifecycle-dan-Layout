# Activity-Lifecycle-dan-Layout
Lakukanlah kompilasi kemudian simpulkan apakah perbedaan antara wrap_content dan match_parent Berikut perbedaan antara wrap_content dan match_parent
Wrap_content digunakan untuk width (lebar) dan height (tinggi) mengikuti object dalam komponen yang ada didalamnya
Match_parent digunakan untuk width (lebar) dan height (tinggi) mengikuti screen pada perangkat device
Berdasarkan percobaan diatas lakukan lah perubahan perubahan sebagai berikut kemudian ambil lah kesimpulan dari percobaan ini.
ubah orientasi menjadi horizontal Ketika orientasi diubah menjadi horizontal maka per object akan ditambahkan secara horizontal.
ubah weight sum menjadi angka integer yang lain Weight sum dapat mendefinisikan weight menggunakan layout_weight
ubah layout_weight dari salah satu LinearLayout Suatu object memiliki ukuran yang tergantung kepada weight_sum dan layout_weight

Kesimpulan
Dari percobaan diatas dapat disimpulkan bahwa output menjadi horizontal. Weight_sum dan layout_weight harus memiliki perbandingan yang seimbang,maka akan mendapat output yang sama, ketika perbandingan weight_sum dan layout_weight tidak seimbang maka akan mendapatkan output yang tidak sama. Karena keduanya memiliki ketergantungan.
